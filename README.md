# About the project

G-EDM provides Hard- and Software solution for DIY sinker and wire EDM machining targeting the hobby market while trying to keep the project affordable and low budget compatible. While this is one of the main goals there are no compromises made in regards of precision and machine rigidity.

The project evolved over time and so did the mechanical design, the electronics and the firmware.

This repo acts as a gateway to all the other repos and is a way to start the journey into the world of DIY electrical discharge machining.

There are currently two different core version of the electronics and firmware in circulation. The EVOII/III are the initial releases. The new Gapstorm components are the current and upcoming versions. 

Since the sensing circuitry has slightly changed the Gapstorm firmware is not compatible with the EVO pulseboard.

It is possible to upgrade the EVOII and III Pulseboard to match the Gapstorms. A tutorial on how to do that will follow soon. After that upgrade those baords can be used with the new firmware.

# Electronics

The BOM for the control unit is included on this repo:

[BOM for the control unit](./control-unit-BOM.md)



Files for the PCBs are available on PCBway. To build the electronics it requires two custom PCBs. The switching PCB and the Controlboard aka Motionboard. 

The current PCBs are the Gapstorm Pulseboard and the EVOIII Motionboard.


[Gapstorm Pulseboard on PCBway](https://www.pcbway.com/project/shareproject/G_EDM_Gapstorm_Pulseboard_14dd4341.html)


[EVOIII Motionboard on PCBway](https://www.pcbway.com/project/shareproject/W532897ASC41_motionboard_evoIII_rev3_bb113e0f.html)


</br>
